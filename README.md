# My First Portfolio

This project is my first HTML website for the ALX Front-End program.  
It teaches the basics of creating a simple webpage using only HTML.

## Project Structure
- html_basic/
    - base_index.html
- README.md

## What I Learned
- How to create a basic HTML page
- How to use DOCTYPE, head, and body tags
- How to validate HTML with the W3C validator
- How to organize files inside folders

## Requirements
- Use only HTML/CSS/JavaScript (no external libraries)
- All files must end with a new line
- Code must be W3C compliant
- Must pass the W3C HTML and CSS validators
